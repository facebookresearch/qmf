# Contributing to qmf
This repo exists to serve as the example code for our paper, "QMF-Blend: Quantized Matrix Factorization for Efficient
Blendshape Compression".  We do not expect a lot of active development internal to Meta; however, we will accept bug fixes and improvements from the community.

## Our Development Process
We do not expect much active development on the project.

## Pull Requests
We welcome pull requests.

1. Fork the repo and create your branch from `main`.
2. Ensure you've tested with scenes in the repo.
3. If you haven't already, complete the Contributor License Agreement ("CLA").

## Contributor License Agreement ("CLA")
In order to accept your pull request, we need you to submit a CLA. You only need
to do this once to work on any of Meta's open source projects.

Complete your CLA here: <https://code.facebook.com/cla>

## Issues
We use GitHub issues to track public bugs. Please ensure your description is
clear and has sufficient instructions to be able to reproduce the issue.

Meta has a [bounty program](https://www.facebook.com/whitehat/) for the safe
disclosure of security bugs. In those cases, please go through the process
outlined on that page and do not file a public issue.

## Coding Style  
Please attempt to follow surrounding code conventions when making changes.
